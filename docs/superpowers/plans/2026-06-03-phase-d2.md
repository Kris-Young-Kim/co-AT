### 1. 의존성 설치
- `@react-pdf/renderer` hr 앱에 추가

- `hr_employees`: `bank_name TEXT`, `bank_account TEXT` 추가

### 4. 사이드바 급여관리 카테고리 확장
현재: 급여관리, 일용급여
추가: 지급공제항목, 급상여생성, 급여대장, 급여명세서, 급여지급현황, 급여계좌이체명세서, 근로소득원천징수부

### 5. 신규 페이지/컴포넌트
- `/salary/pay-items` — 지급공제항목 등록
- `/salary/generate` — 급상여생성 (월별 일괄)
- `/salary/ledger` — 급여대장 (월 선택 → 전체 직원)
- `/salary/slips` — 사원별 급여명세서 목록
- `/salary/[yearMonth]/slip/[employeeId]` — 급여명세서 상세 + PDF 다운로드
- `/salary/payment-status` — 급여지급현황 (월별 요약)
- `/salary/transfer-list` — 급여계좌이체명세서
- `/salary/withholding-tax` — 근로소득원천징수부 (연간)

### 6. PDF 컴포넌트
- `SalarySlipPDF.tsx` — @react-pdf/renderer 기반 급여명세서
