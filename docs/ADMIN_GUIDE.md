## 1. 관리자 로그인하기

### 1.1 관리자 페이지 들어가기

웹사이트 주소 뒤에 `/admin`을 붙여서 들어갑니다.

예: `https://co-at.example.com/admin`

### 1.2 로그인하기

일반 사용자와 동일하게 로그인합니다:
- 이메일과 비밀번호
- 구글 로그인
- 카카오 로그인

> ⚠️ **중요**: 관리자 권한이 있는 계정으로만 로그인할 수 있어요!

### 1.3 관리자 권한 확인

로그인 후 관리자 권한이 있으면 자동으로 관리자 페이지로 이동합니다.
권한이 없으면 일반 사용자 페이지로 이동합니다.

## 2. 대시보드 보기

### 2.1 대시보드란?

대시보드는 **오늘의 업무를 한눈에 볼 수 있는 화면**입니다!

```
┌─────────────────────────────────────┐
│  📊 통합 대시보드                   │
├─────────────────────────────────────┤
│  오늘의 신규 접수: 5건              │
│  진행 중: 12건                      │
│  오늘 완료: 3건                     │
│                                     │
│  [5대 사업별 현황]                  │
│  💬 상담: 10건                      │
│  🎮 체험: 5건                       │
│  🛠️ 맞춤형: 8건                     │
│  🔧 사후관리: 15건                  │
│  📚 교육: 3건                       │
└─────────────────────────────────────┘
```

### 2.2 대시보드에서 할 수 있는 것

- **오늘의 신규 접수**: 오늘 들어온 새로운 신청서 확인
- **진행 중인 업무**: 지금 처리하고 있는 신청서 확인
- **오늘의 일정**: 오늘 해야 할 일 확인
- **5대 사업 현황**: 각 사업별로 얼마나 많은 신청이 있는지 확인

## 3. 신청서 관리하기

### 3.1 신청서 목록 보기

왼쪽 메뉴에서 **"대상자 관리"** 또는 **"신청서"**를 클릭합니다.

### 3.2 신청서 상태 변경하기

신청서를 클릭하면 상세 내용을 볼 수 있습니다.

**상태 변경 방법:**
1. 신청서 상세 페이지에서 **"상태 변경"** 버튼 클릭
2. 원하는 상태 선택:
   - **접수**: 신청서가 들어왔어요
   - **진행**: 지금 처리하고 있어요
   - **완료**: 모든 작업이 끝났어요
   - **취소**: 신청이 취소되었어요
3. **"저장"** 버튼 클릭

### 3.3 담당 직원 배정하기

1. 신청서 상세 페이지에서 **"담당자 배정"** 버튼 클릭
2. 담당할 직원 선택
3. **"저장"** 버튼 클릭

> 💡 **팁**: 담당자를 배정하면 해당 직원에게 알림이 갑니다!

### 3.4 신청서 검색하기

많은 신청서 중에서 원하는 것을 찾으려면:
1. 검색창에 키워드 입력 (이름, 기기명 등)
2. 필터 사용 (상태, 날짜, 사업 종류)
3. **"검색"** 버튼 클릭

## 4. 대상자 관리하기

### 4.1 대상자 목록 보기

왼쪽 메뉴에서 **"대상자 관리"**를 클릭합니다.

### 4.2 대상자 정보 보기

대상자 이름을 클릭하면 상세 정보를 볼 수 있습니다:

```
┌─────────────────────────────────────┐
│  대상자 상세 정보                   │
├─────────────────────────────────────┤
│  이름: 홍길동                       │
│  생년월일: 1980-01-01               │
│  연락처: 010-1234-5678              │
│  주소: 강원도 춘천시...             │
│                                     │
│  [서비스 이력]                      │
│  - 2025-01-15: 수리 신청 (완료)     │
│  - 2025-01-20: 대여 신청 (진행 중)  │
│                                     │
│  [상담 기록]                        │
│  - 2025-01-10: 전화 상담            │
└─────────────────────────────────────┘
```

### 4.3 대상자 정보 수정하기

1. 대상자 상세 페이지에서 **"정보 수정"** 버튼 클릭
2. 수정할 내용 입력
3. **"저장"** 버튼 클릭

> ⚠️ **주의**: 중요한 정보를 수정할 때는 신중하게 확인하세요!

### 4.4 상담 기록 작성하기

1. 대상자 상세 페이지에서 **"상담 기록"** 탭 클릭
2. **"새 상담 기록"** 버튼 클릭
3. 상담 내용 입력:
   - 상담 날짜
   - 상담 유형 (전화, 방문, 센터)
   - 상담 내용
4. **"저장"** 버튼 클릭

> 💡 **팁**: AI가 상담 내용을 자동으로 정리해드립니다!

## 5. 재고 관리하기

### 5.1 재고 목록 보기

왼쪽 메뉴에서 **"재고 관리"**를 클릭합니다.

### 5.2 재고 추가하기

1. **"재고 추가"** 버튼 클릭
2. 정보 입력:
   - 기기 이름 (예: 전동휠체어)
   - 카테고리 (예: 이동보조기)
   - 상태 (예: 보관, 대여중, 수리중)
   - 구입일, 가격 등
3. **"저장"** 버튼 클릭

### 5.3 재고 상태 변경하기

1. 재고 목록에서 기기를 클릭
2. **"상태 변경"** 버튼 클릭
3. 상태 선택:
   - **보관**: 창고에 보관 중
   - **대여중**: 누군가 빌려가고 있음
   - **수리중**: 수리를 받고 있음
   - **폐기**: 더 이상 사용할 수 없음
4. **"저장"** 버튼 클릭

### 5.4 재고 검색하기

많은 재고 중에서 찾으려면:
- 검색창에 기기 이름 입력
- 필터 사용 (카테고리, 상태)
- **"검색"** 버튼 클릭

## 6. 일정 관리하기

### 6.1 일정 목록 보기

왼쪽 메뉴에서 **"일정 관리"**를 클릭합니다.

### 6.2 일정 추가하기

1. **"일정 추가"** 버튼 클릭
2. 정보 입력:
   - 일정 종류 (방문, 상담, 평가, 배송 등)
   - 날짜와 시간
   - 대상자 선택
   - 담당 직원 선택
   - 주소 (방문인 경우)
   - 메모
3. **"저장"** 버튼 클릭

### 6.3 캘린더 보기

일정을 **캘린더 형식**으로 볼 수 있어요:
- 월별 보기: 한 달의 일정을 한눈에
- 주별 보기: 한 주의 일정을 자세히
- 일별 보기: 하루의 일정을 시간순으로

### 6.4 일정 수정/삭제하기

1. 캘린더에서 일정을 클릭
2. **"수정"** 또는 **"삭제"** 버튼 클릭
3. 확인 후 저장

## 7. 통계 보기

### 7.1 통계 대시보드 들어가기

왼쪽 메뉴에서 **"통계 대시보드"**를 클릭합니다.

### 7.2 통계 보는 방법

통계 대시보드에서는 다음을 볼 수 있어요:

**전체 요약**
- 전체 신청 건수
- 대상자 수
- 완료 건수
- 완료율

**5대 사업별 통계**
- 각 사업별로 얼마나 많은 신청이 있는지
- 월별/연도별 추이 그래프
- 사업별 비율 파이 차트

### 7.3 기간 선택하기

1. **"기간 선택"** 섹션에서 원하는 기간 선택
2. 월별 통계: 특정 연도 선택
3. 연도별 통계: 시작 연도와 종료 연도 선택
4. 자동으로 통계가 업데이트됩니다!

### 7.4 Excel로 내보내기

통계를 **Excel 파일**로 다운로드할 수 있어요:

1. **"Excel 내보내기"** 버튼 클릭
2. 파일이 자동으로 다운로드됩니다
3. Excel로 열어서 상위 기관에 제출할 수 있어요!

> 💡 **팁**: 지자체 예산 확보 자료로 활용하세요!

## 8. 자주 묻는 질문

### Q1. 관리자 권한을 받으려면 어떻게 해야 하나요?

**A:** 기존 관리자에게 요청하거나, 시스템 관리자에게 문의하세요.
권한은 다음 중 하나입니다:
- **admin**: 모든 권한
- **manager**: 관리 권한
- **staff**: 직원 권한

### Q2. 신청서를 삭제할 수 있나요?

**A:** 일반적으로 삭제는 권장하지 않습니다.
대신 **"취소"** 상태로 변경하세요.
이렇게 하면 기록이 남아서 나중에 확인할 수 있어요.

### Q3. 여러 신청서를 한 번에 처리할 수 있나요?

**A:** 네! 목록에서 여러 개를 선택하고 일괄 처리할 수 있어요:
- 여러 개 선택: 체크박스 클릭
- 일괄 상태 변경: "일괄 처리" 버튼 클릭

### Q4. 통계는 언제 업데이트되나요?

**A:** 실시간으로 업데이트됩니다!
새로운 신청서가 들어오면 바로 통계에 반영돼요.

### Q5. Excel 파일을 상위 기관에 제출해도 되나요?

**A:** 네! 통계 대시보드에서 내보낸 Excel 파일은 상위 기관 양식에 맞춰 만들어졌어요.
그대로 제출하시면 됩니다!

### Q6. 실수로 잘못된 정보를 입력했어요 😢

**A:** 걱정 마세요! 대부분의 정보는 수정할 수 있어요.
- 수정 버튼을 클릭
- 올바른 정보로 수정
- 저장

다만, 이미 완료된 서비스는 수정이 제한될 수 있어요.

### Q7. 다른 직원이 한 일을 확인할 수 있나요?

**A:** 네! 각 신청서와 기록에는 **담당자**와 **작성자** 정보가 표시됩니다.
누가 언제 무엇을 했는지 모두 확인할 수 있어요!
