## 1. 웹사이트에 들어가기

### 1.1 인터넷 주소 입력하기

컴퓨터나 스마트폰의 **인터넷 브라우저**를 엽니다.
- 컴퓨터: 크롬(Chrome), 엣지(Edge), 사파리(Safari) 등

> 💡 **팁**: 주소를 외우기 어렵다면 즐겨찾기에 저장해두세요!

## 2. 회원가입하기

### 2.1 회원가입 버튼 찾기

화면 오른쪽 위에 있는 **"로그인"** 또는 **"회원가입"** 버튼을 클릭합니다.

### 2.2 간편 로그인으로 가입하기

회원가입은 **매우 쉽습니다!** 다음 중 하나를 선택하면 됩니다:

- 📧 **이메일**: 이메일 주소와 비밀번호 입력
- 🍎 **구글**: 구글 계정으로 로그인
- 💬 **카카오**: 카카오톡 계정으로 로그인

> 💡 **팁**: 구글이나 카카오로 가입하면 비밀번호를 외울 필요가 없어요!

### 2.3 정보 입력하기

이메일로 가입하는 경우:
1. 이름 입력
2. 이메일 주소 입력 (예: `hong@example.com`)
3. 비밀번호 입력 (8자 이상, 숫자와 영문 포함)
4. 비밀번호 다시 입력 (확인용)
5. **"가입하기"** 버튼 클릭

> ⚠️ **주의**: 비밀번호는 다른 사람이 알 수 없도록 안전하게 만들어주세요!

## 3. 로그인하기

### 3.1 로그인 버튼 클릭

화면 오른쪽 위의 **"로그인"** 버튼을 클릭합니다.

### 3.2 로그인 방법 선택

회원가입할 때 사용한 방법을 선택합니다:
- 이메일로 가입했다면 → 이메일과 비밀번호 입력
- 구글로 가입했다면 → 구글 로그인 버튼 클릭
- 카카오로 가입했다면 → 카카오 로그인 버튼 클릭

### 3.3 로그인 완료

로그인이 성공하면 화면 오른쪽 위에 **내 이름**이 보입니다!

> 💡 **팁**: "로그인 상태 유지"를 체크하면 다음에 자동으로 로그인됩니다.

## 4. 서비스 신청하기

### 4.1 서비스란?

보조기기센터에서 제공하는 **5가지 서비스**가 있습니다:

1. **상담 및 정보제공** 💬
   - 전화 상담, 방문 상담, 견학
   - 보조기기에 대해 궁금한 것을 물어볼 수 있어요!

2. **체험** 🎮
   - 보조기기를 직접 사용해볼 수 있어요
   - 마음에 들면 대여할 수도 있습니다!

3. **맞춤형 지원** 🛠️
   - 보조기기 대여
   - 나에게 딱 맞는 보조기기 제작
   - 보조기기 평가 지원

4. **사후관리** 🔧
   - 보조기기 수리
   - 소독 및 세척
   - 재사용 지원

5. **교육 및 홍보** 📚
   - 보조기기 사용법 교육
   - 홍보 행사 참여

### 4.2 서비스 신청하기

#### 단계 1: 서비스 선택

1. 화면 상단의 **"서비스 신청"** 또는 **"신청하기"** 버튼을 클릭합니다.
2. 원하는 서비스를 선택합니다.

```
┌─────────────────────────────────────┐
│  어떤 서비스를 원하시나요?          │
├─────────────────────────────────────┤
│  [💬 상담]  [🎮 체험]  [🛠️ 맞춤형] │
│  [🔧 사후관리]  [📚 교육]          │
└─────────────────────────────────────┘
```

#### 단계 2: 세부 내용 입력

선택한 서비스에 따라 필요한 정보를 입력합니다:

**예시: 수리 신청**
- 기기 이름: (예: 휠체어)
- 기기 모델: (예: ABC-123)
- 문제 설명: (예: 바퀴가 잘 안 돌아요)
- 사진 첨부: (문제가 있는 부분 사진)

**예시: 대여 신청**
- 대여할 기기: (예: 전동휠체어)
- 대여 시작일: (예: 2025년 2월 1일)
- 대여 종료일: (예: 2025년 2월 28일)
- 대여 목적: (예: 여행을 가기 위해 필요해요)

> 💡 **팁**: 모르는 부분이 있으면 일단 입력하고, 나중에 수정할 수 있어요!

#### 단계 3: 확인 및 제출

1. 입력한 내용을 다시 한 번 확인합니다.
2. 모든 내용이 맞다면 **"제출하기"** 버튼을 클릭합니다.
3. "신청이 완료되었습니다!" 메시지가 나오면 성공입니다! 🎉

## 5. 내 신청서 확인하기

### 5.1 마이페이지 들어가기

로그인 후 화면 오른쪽 위에 있는 **내 이름**을 클릭합니다.
또는 **"마이페이지"** 메뉴를 클릭합니다.

### 5.2 신청서 목록 보기

마이페이지에서 내가 신청한 모든 서비스를 볼 수 있습니다:

```
┌─────────────────────────────────────┐
│  내 신청서 목록                     │
├─────────────────────────────────────┤
│  📋 수리 신청 (2025-01-15)          │
│     상태: 진행 중 🔄                │
│                                     │
│  📋 대여 신청 (2025-01-20)          │
│     상태: 접수 완료 ✅              │
│                                     │
│  📋 상담 신청 (2025-01-25)          │
│     상태: 완료 🎉                   │
└─────────────────────────────────────┘
```

### 5.3 상태 확인하기

신청서는 다음과 같은 상태로 표시됩니다:

- **접수** 📝: 신청서가 접수되었어요
- **진행 중** 🔄: 센터에서 처리하고 있어요
- **완료** ✅: 서비스가 완료되었어요
- **취소** ❌: 신청이 취소되었어요

### 5.4 상세 내용 보기

신청서를 클릭하면 자세한 내용을 볼 수 있습니다:
- 신청 날짜
- 담당 직원
- 진행 상황
- 완료 예정일

## 6. 자주 묻는 질문

### Q1. 비밀번호를 잊어버렸어요 😢

**A:** 로그인 화면에서 **"비밀번호 찾기"** 버튼을 클릭하세요.
이메일로 비밀번호 재설정 링크를 보내드립니다.

### Q2. 신청서를 수정하고 싶어요

**A:** 아직 접수 상태라면 수정할 수 있어요!
마이페이지에서 신청서를 클릭하고 **"수정하기"** 버튼을 누르세요.

### Q3. 신청을 취소하고 싶어요

**A:** 마이페이지에서 신청서를 클릭하고 **"취소하기"** 버튼을 누르세요.
다만, 이미 진행 중인 서비스는 취소가 어려울 수 있어요.

### Q4. 언제까지 기다려야 하나요?

**A:** 서비스 종류에 따라 다릅니다:
- 상담: 1-2일
- 수리: 3-5일
- 대여: 1-2일
- 맞춤 제작: 1-2주

정확한 일정은 담당 직원이 연락드립니다!

### Q5. 전화로 신청할 수 있나요?

**A:** 네! 전화로도 신청할 수 있어요.
센터 전화번호: 1670-5529

### Q6. 스마트폰에서도 사용할 수 있나요?

**A:** 네! 스마트폰, 태블릿, 컴퓨터 모두 사용할 수 있어요!
어디서든 편리하게 신청할 수 있습니다.
